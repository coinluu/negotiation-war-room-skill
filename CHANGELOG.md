## [0.1.0] - 2026-07-01

### Added

- 创建 `negotiation-war-room-skill` 初始版本。
- 增加核心协议文件 `skill.md`。
- 增加系统提示词、输入模板、输出模板、模拟提示词、风险审查与复盘提示词。
- 增加谈判框架：BATNA、ZOPA、Anchoring、Interest-based negotiation、MESO、Trade-off、Loss Aversion、Commitment。
- 增加权力指数、场景分流、让步规则、MESO 多方案生成规则和风险清单。
- 增加 8 个场景 Playbook：尾款结算、客户压价、薪资谈判、合同条款、合伙谈判、纠纷协商、采购销售、关系沟通。
- 增加微信、邮件、电话、会议、谈判纪要和跟进模板。
- 增加完整案例：尾款结算、客户压价、薪资谈判、合伙谈判。
- 增加测试 Prompt、边界案例和验收清单。
- 增加 GitHub 社区协作文件：Issue 模板、PR 模板、贡献指南、安全政策和行为准则。

### Security

- 明确禁止欺骗、威胁、敲诈、伪造证据、违法施压、冒充律师和恶意操控。
- 明确不承诺保证谈判成功。
- 明确法律、劳动仲裁、合同争议只提供沟通策略、证据整理和风险提示。
