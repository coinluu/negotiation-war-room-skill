# 熟人合作、家庭关系、亲密关系沟通 Playbook

## 1. 场景定义

适用于熟人合作、朋友项目、亲戚借钱、家庭分工、伴侣沟通、经济边界、关系修复和亲密关系冲突。核心不是赢过对方，而是在情绪安全下表达需求、边界和可执行共识。

## 2. 常见用户处境

- 朋友或亲戚提出合作、借钱、帮忙，用户不好拒绝。
- 家庭成员长期让用户承担经济或情绪压力。
- 伴侣或家人沟通容易情绪化。
- 用户既想维护关系，又不想继续委屈自己。
- 熟人合作没有合同，后续出现利益争议。

## 3. 用户常见错误

- 为了面子答应超出承受范围的请求。
- 一直忍耐，最后爆发。
- 用商业压价、威胁、冷处理方式处理亲密关系。
- 只讲道理，不承接情绪。
- 借钱或合作没有金额、时间和责任边界。

## 4. 谈判目标

- 优先目标：表达真实需求和边界，降低冲突。
- 次级目标：达成可执行共识，例如金额、时间、分工、频率、规则。
- 底线目标：不牺牲安全、尊严、基本经济边界和长期心理承受能力。

## 5. 关键风险

- 情绪升级，关系受损。
- 用户因内疚继续无边界让步。
- 对方把边界理解为拒绝关系。
- 涉及威胁、控制、暴力或人身安全。
- 熟人合作因不留痕导致后续争议。

## 6. 信息采集重点

```text
关系类型：
当前冲突：
用户真实需求：
用户不能接受的边界：
对方可能感受：
对方实际诉求：
是否涉及金钱/合作/照护/同住：
是否存在威胁、控制或暴力：
希望维持到什么关系状态：
沟通渠道：微信/电话/面谈
```

## 7. 权力指数判断重点

- 替代方案：用户是否可以暂停、拒绝、求助第三方或设置距离。
- 时间压力：谁更急于得到回应、钱、帮助或承诺。
- 信息优势：用户是否清楚自己的需求、边界和事实。
- 资源控制：谁控制经济、居住、照护、关系资源。
- 证据强度：熟人合作是否有聊天、转账、约定记录。
- 关系依赖：用户是否因亲情、伴侣关系或面子过度让步。

## 8. 筹码分析重点

- 情绪筹码：尊重、理解、被看见。
- 边界筹码：时间、金钱、精力、空间。
- 关系筹码：继续沟通、一起解决、保留体面。
- 规则筹码：借款日期、还款计划、合作分工、家庭分工。
- 安全筹码：第三方陪同、公开场合、专业帮助。

## 9. 推荐策略

1. 先确认关系和情绪，再表达具体事实。
2. 使用“事实 - 感受 - 需求 - 请求”结构。
3. 边界要具体，不做人格攻击。
4. 熟人合作或借钱必须写明金额、时间、责任。
5. 涉及威胁、暴力、控制时，优先安全，不建议单独谈判。

## 10. 不建议动作

- 不建议用“你总是/你从来不”。
- 不建议用冷暴力、威胁、报复或羞辱。
- 不建议为了关系答应无法承受的事。
- 不建议把商业强压话术套到家庭关系。
- 不建议在人身安全风险下独自面谈。

## 11. 第一轮话术模板

```text
我很重视我们的关系，所以这件事我不想靠猜、忍或者突然爆发来处理。

我想具体说一下：最近让我压力比较大的是 A。我的感受是 B，我真正需要的是 C。

我愿意在 D 上配合，但 E 是我的边界。我们能不能先按这个方式试一段时间，然后在 X 时间再复盘？
```

## 12. 对方常见反应

- “你怎么这么计较？”
- “都是一家人/朋友，谈这些太生分。”
- “你就是不在乎我。”
- “以前都可以，为什么现在不行？”
- “那算了，以后别来往了。”

## 13. 应对话术

对方说你计较：

```text
我不是想计较，也不是否定我们的关系。我是希望把这件事说清楚，避免我一直压着，最后反而影响关系。
```

对方说谈规则生分：

```text
正因为关系重要，才更需要把边界说清楚。规则不是疏远，是为了减少误会。
```

对方情绪化：

```text
我听到你现在很不舒服。我们可以先暂停一下，但这件事我不想继续回避。等情绪缓一点，我们再回到具体怎么做。
```

## 14. 谈后确认文本

```text
我整理一下我们刚才说到的内容：
1. 我在意的是我们的关系，也希望事情能更清楚地处理。
2. 目前让我有压力的是：……
3. 我可以做到：……
4. 我的边界是：……
5. 我们先按……执行到 X 时间，再一起复盘。

如果你觉得我理解有偏差，也可以补充。
```

## 15. 复盘问题

- 对方是否听懂了用户的具体边界？
- 用户是否因为内疚又做了超出承受范围的承诺？
- 这次沟通是降低了冲突，还是升级了风险？
- 是否需要第三方、家庭成员、朋友或专业人士介入？
- 是否存在安全风险，需要优先离开或求助？
