# 安全边界

## 项目定位

本项目提供谈判策略、沟通结构、话术草案、风险审查和复盘问题。它不提供法律意见，不替代律师、仲裁员、心理咨询师、财务顾问或其他专业人士。

## 禁止生成

- 欺骗、威胁、敲诈、骚扰、违法施压。
- 伪造聊天记录、合同、付款记录、证据或身份。
- 冒充律师、监管机构、HR、客户或第三方。
- 情绪勒索、恶意操控、羞辱、报复、隐私侵犯。
- 保证谈判成功、一定追回、一定胜诉等承诺。

## 高风险场景

以下场景只能做沟通策略、证据整理和风险提示：

- 合同争议、违约、赔偿。
- 劳动仲裁、竞业限制、离职补偿。
- 诉讼、监管投诉、行政处罚。
- 家庭财产、人身安全、暴力、威胁、控制。
- 隐私、名誉、知识产权或数据合规。

## 推荐安全表达

```text
我不能替代律师给出法律结论，但可以帮你整理事实、证据、风险点和沟通策略。涉及正式权利义务判断或可能仲裁/诉讼的部分，建议让律师审阅后再发送。
```
